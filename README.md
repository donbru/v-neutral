## Requirements
1. Install node.js

## Installation 

1. Clone git project
1. ```cd 
v-neutral
```
1. ```npm
install
```
-- node runs *package.json*, creates folder *node_modules*, and install module dependencies.
1. ```node
app.js
```
-- runs node application *app.js*.
1. Browse to "http://localhost:3000/"
